1. To use and see this example first create a database with the name pagination in phpmyadmin.
2. Inside pagination database create a table with the name students_directory and with the following columns(name,student_code,address,hobby)
3. import the "pagination.sql" database to phpmyadmin by clicking on the import options available in your phpmyadmin when you select the created database.
4. Insert data and Check the working of the database.
